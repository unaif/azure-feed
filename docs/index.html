Azure IP Ranges — Feed sanitizado

Endpoint público para consumir como External Dynamic List (Palo Alto) o External Threat Feed (FortiGate).

Origen: Microsoft Azure IP Ranges and Service Tags
Refresco: cada lunes vía GitHub Actions (Microsoft publica actualizaciones los lunes).
Formato: una entrada por línea, CIDRs en notación CIDR estándar.

Feeds combinados (uso típico)

  • azure_ipv4.txt — Todos los CIDRs IPv4
  • azure_ipv6.txt — Todos los CIDRs IPv6

Feeds por servicio (IPv4)

  • azure_frontdoor_ipv4.txt — AzureFrontDoor.*
  • azure_storage_ipv4.txt — Storage.*
  • azure_cdn_ipv4.txt — AzureCDN.*
  • azure_sql_ipv4.txt — Sql.*

Metadatos

  • meta.json — timestamp del último sync, changeNumber y conteos por servicio
